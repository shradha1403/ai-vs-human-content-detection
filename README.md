AI vs. Human Content Detection: A Multilingual Analysis

This project analyzes the AI vs. Human Content Detection Dataset (2026 Edition) to explore how AI-generated, human-written, and post-edited AI content differ across domains, languages, and text length.

Research Questions





How do word counts differ between content types, and does domain amplify those differences?



Which content types dominate each domain across all three labels?



How does language interact with content type and word length?

Key Findings





Word count is the strongest single detection signal. Human-written content has a median of 33 words vs. 347 for AI-generated



No domain is safe from AI content. AI makes up 45-51% across all six domains



Language matters. Hindi and English have the highest AI rates (60% and 57%), while Spanish, French, and code-mixed content are lowest

Tools Used





Python (pandas, matplotlib, seaborn)



R (ggplot2, tidyverse)

Dataset Publicly available on Kaggle: https://www.kaggle.com/datasets/mafaqbhatti/ai-vs-human-content-detection-dataset-2026
